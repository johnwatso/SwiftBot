SwiftBot 1.20

Version 1.20 · Build 2026052301

The most significant update to SwiftBot yet. Version 1.20 introduces a massive security overhaul, hardened clustering with SwiftMesh, a powerful new Sweep subsystem for server health, and a complete migration to Swift 6.

🛡️ Security & Auditing

Advanced protection for your bot and transparency for your team.

  • Content Security Policy (CSP): Implemented a strict CSP with per-request nonces for the Admin Web UI. All inline JavaScript has been removed in favor of a delegated event dispatcher, mitigating XSS risks.
  • Mandatory Discord MFA: Web UI login now requires Multi-Factor Authentication to be enabled on your Discord account.
  • Structured Audit Log: A new persistent log for tracking authentication, configuration changes, and moderation actions (including those triggered by Automations and Sweep).
  • Keychain Session Storage: All administrative sessions are now stored in the secure macOS Keychain, encrypted at rest.
  • IP-Based Rate Limiting: Hardened against brute-force attacks with failure-window lockouts keyed by peer IP and username.
  • Cleartext Blocking: The Admin Web Server now refuses to serve credentials or cookies over plain HTTP on routable addresses; HTTPS is mandatory for non-loopback access.
  • Secure Media Tokens: Recording and stream playback now use short-lived, HMAC-signed tokens instead of persistent session cookies.
  • Session Binding: Sessions are bound to the browser's User-Agent to mitigate session hijacking.

🕸️ SwiftMesh & Clustering

Bulletproof high-availability for enterprise bot deployments.

  • AES-GCM Encryption: All SwiftMesh traffic between nodes is now encrypted using AES-GCM keyed off your shared secret.
  • Hardened Failover: Added split-brain protection, confirmed-retry logic, and cursor preservation during leader handovers.
  • Auto-Reclaim: Configured Primary nodes can now automatically reclaim leadership once they recover from an outage.
  • Cluster Map Overhaul: A compact, high-fidelity cluster map in the Overview tab with custom node-icon overrides and live latency tracking.
  • Handover Testing: New connection diagnostic tools and handover simulation to verify your cluster's resilience.

🧹 Sweep & Automations

Keeping your server healthy and your workflows automated.

  • Sweep Subsystem: A new background engine that retroactively scans your server for missing permissions, inactive roles, and security misconfigurations.
  • Dry-Run Previews: See exactly what a Sweep policy or automation rule will do before it applies changes.
  • Welcome Flow v2: A complete refactor of the member join/leave service with support for rich embeds, invite tracking, and account age gating.
  • New /steam Command: Direct integration with the Steam Store for game lookups and metadata embedding.
  • Lookup Service: WikiBridge has been expanded into a unified Lookup engine for Wikipedia, Steam, and GitHub.

✨ UI & Experience

Redesigned from the ground up for performance and clarity.

  • UI Consistency: A comprehensive visual refresh across all native views and the admin web interface, ensuring a seamless and consistent experience throughout the application.
  • Unified Activity Log: Merged Command Log and System Logs into a single, high-performance view with smart filtering for Gateway, Audit, and Mesh events.
  • Premium Login Experience: A completely redesigned, Apple-inspired login screen for the Web UI with high-density background animations.
  • Lucide Icon Migration: Replaced legacy icon sets with modern, consistent Lucide icons across both native and web interfaces.
  • Variable Autocomplete: Intelligent autocomplete for template variables in the Web UI automation editor.
  • Metric Grid: Redesigned dashboard cards with clearer trends and balanced layouts.

⚙️ Core Infrastructure

The foundation for the next generation of SwiftBot.

  • Swift 6.0 Migration: The entire codebase now leverages strict concurrency checking for maximum stability and performance.
  • Voice Overhaul: Comprehensive updates to Opus encoding and UDP transport for improved TTS and playback reliability.
  • Recording Playback: Significant performance improvements to the recording playback and streaming engine.
  • Timezone Support: Native support for local timezones and dynamic timestamp formatting in all command responses.

📖 Documentation

Helping you get the most out of SwiftBot.

  • Bot Setup Guide: A comprehensive walkthrough covering tokens, intents, OAuth, and invites.
  • Web UI Setup Guide: Detailed instructions for local exposure, including a dedicated Cloudflare Tunnel guide.
