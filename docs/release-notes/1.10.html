SwiftBot 1.10

Version 1.10 · Build 2026050801

Version 1.10 marks the official introduction of the SwiftMiner integration ecosystem and the all-new Analytics Engine. This update brings a massive amount of UI polish, performance improvements, and architectural cleanup.

⛏️ SwiftMiner Integration

Seamlessly pair your bot with SwiftMiner for advanced notifications.

  • Unified Pairing: Added a dedicated SwiftMiner tab for effortless pairing, complete with status cards and connection health indicators.
  • Onboarding DMs: Implemented a setup flow that sends one-click activation URLs via Discord DM for streamlined Twitch account linking.
  • Real-time Webhooks: Native support for SwiftMiner webhooks, enabling the bot to relay mining status and reward notifications instantly.
  • Durable Messaging: Added persistent DM deduplication and frequency controls to prevent notification bombardment during high-activity periods.
  • Pairing Server: Enabled a dedicated background pairing server to handle automated handshakes between SwiftBot and SwiftMiner instances.

📊 Analytics & Monitoring

Deeper insights into your bot's performance and server activity.

  • New Dashboards: Introduced high-fidelity analytics dashboards in both the native macOS app and the Admin Web UI.
  • Voice Insights: Rebalanced the Voice Analytics view with improved insights into user participation and streaming trends.
  • Smart Heartbeats: Softened the 'Event flow stalled' notice into a more intelligent 'Runtime feed quiet' alert that requires verified inactivity before firing.
  • Refined Metrics: Memory usage metrics now utilize a rolling 3-minute window and render in whole megabytes for better readability.

✨ UI & UX Polish

A round of premium refinements across the entire interface.

  • Sidebar Refresh: Redesigned the sidebar for better density and cleaner navigation.
  • Integrations Tab: Restyled with grouped forms and status cards to match the design language of the SwiftMiner ecosystem.
  • Performance Boosts: Optimized chart rendering and reduced expensive shadow effects on operational cards to eliminate scroll lag.
  • Native Views: Massive overhaul of the RootView and layout engine for better resizing behavior and visual balance.

⚙️ Core & Infrastructure

Hardening the foundation for a more stable experience.

  • Decomposition: Major architectural cleanup, splitting the monolithic Models.swift into smaller, domain-specific components.
  • App-Owned Security: Clarified app-owned security components and added a dedicated Security Policy to the project.
  • Patchy Resilience: Hardened the Patchy update engine to better handle AMD-based fetch failures and network churn.
  • Project Cleanup: Switched project license to MIT and removed legacy architecture reports, risk matrices, and obsolete Python scripts.
  • Sandboxing: Pinned user script sandboxing settings in project.yml to prevent build phase issues with SwiftLint.

📖 Documentation

Better guides for setup and security.

  • README Overhaul: Switched to a more condensed, theme-aware README with updated UI previews.
  • Security Policy: Added a formal security policy and clarified vulnerability reporting procedures.
  • SwiftMiner Docs: Integrated detailed documentation for SwiftMiner linking and webhook configuration.
