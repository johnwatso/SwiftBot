SwiftBot 1.20.1

Version 1.20.1 · Build 2026060911

This release focuses on bug fixes for Patchy monitoring and Cloudflare setup.

  • Fixed Patchy showing monitoring as off after enabled targets were added or edited.
  • Removed the extra Patchy Run Now control strip above the targets list.
  • Improved Cloudflare token handling when pasted values include bearer or authorization header text.
  • Improved Cloudflare verification errors so invalid request headers and authentication failures show clearer feedback.
