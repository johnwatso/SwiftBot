Install SwiftBot on macOS

SwiftBot is a native Apple Silicon Mac app. Use the signed GitHub release for day-to-day installs and let Sparkle handle updates from there.

SwiftBot runs entirely on your Mac and connects directly to Discord. Once it is installed, you only need a Discord bot token to get started.

Requirements

  • macOS 26 or later.
  • An Apple Silicon Mac.
  • A Discord bot application and token.
  • Internet access for Discord, updates, and any optional tunnels.

Install steps

  1. Download the release:

    Grab the latest SwiftBot .zip from GitHub Releases.

  2. Move it to Applications:

    Unzip the download and move SwiftBot.app to /Applications.

  3. Launch SwiftBot:

    Open the app from Applications or Spotlight.

  4. Paste your bot token:

    During onboarding, paste your Discord bot token. SwiftBot validates it and stores it in the macOS Keychain.

Don't have a token yet?

Create the Discord application first, then come back here. See Discord Bot Setup for the full walkthrough.

Updates

SwiftBot uses Sparkle for in-app updates. Choose the stable or beta channel under Settings > Updates. Stable is recommended for production servers; beta surfaces new features earlier.
