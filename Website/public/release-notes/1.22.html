SwiftBot 1.22

Version 1.22 · Build 2026062612

A voice-focused release: lower-latency speech, steadier playback, and full visibility into DAVE end-to-end encryption.

🔊Voice Announcer

Speech starts sooner and plays more reliably.

  • Streaming TTS: Synthesis feeds straight into the Opus pipeline to cut playback latency.
  • Steady timing: RTP timestamps realign to wall-clock after idle gaps so resumed audio isn't scheduled late by Discord.
  • Stall watchdog: A fixed playback timeout is replaced by an inactivity watchdog so long messages play in full, and cancelling playback now stops the synthesizer.
  • Stale-credential guard: Tracks the gateway session a voice token belongs to and refuses to connect with stale credentials (close code 4006).
  • Cleaner lifecycle: Clean voice teardown on stop, plus a new /announce rejoin command.

🔐DAVE Diagnostics

End-to-end encryption is now fully observable in-app.

  • Surfaces every DAVE handshake, transition, and roster event to the in-app voice diagnostics log (previously OS-log only), including the version-0 downgrade path.
  • Keeps spoken message content out of the diagnostics log.
